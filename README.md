## Cinema Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/01fba9851b664a6983b6d58dd843dc47/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. View movie information, review movies, select seats and book movies, favorite movies
5. Movie News
6. Personal Center
7. My Orders
8. My Address
9. My Favorites

**Administrator Functions:**
1. Login
2. Personal Center
3. User Management
4. Movie Type Management
5. Cinema Hall Management
6. Movie Information Management
7. Ticket Purchase Statistics Management, Export, Statistical Reports
8. Carousel Management
9. Movie News Management
10. Online Customer Service
11. Order Management
